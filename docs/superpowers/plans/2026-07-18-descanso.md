# Descanso (bloqueio forçado de tela) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans (inline)
> ou subagent-driven-development. Steps usam checkbox (`- [ ]`) pra tracking.

**Goal:** Bloqueio de tela agendado que força uma pausa — cobre todas as telas com
overlay escuro + contador regressivo, em modo quiosque com escape de emergência.

**Architecture:** Reusa o motor de agenda dos Lembretes (generalizado pra
`ScheduleEngine<Item: Scheduled>`). Um `DescansoController` (AppKit) monta uma
janela de shield por tela e liga o quiosque nativo; a view do contador é SwiftUI
puro (snapshot-able). Dois gatilhos: lista própria (aba "Descanso") + pausas do
Pomodoro.

**Tech Stack:** Swift 5/AppKit/SwiftUI/Combine · XcodeGen · `swiftc` self-checks ·
`tools/snapshot.sh`.

**Spec de referência:** `SPEC-descanso.md` (design completo + Achados da pesquisa).

## Global Constraints

- Deployment target macOS 14.2; rodando em macOS 26 Tahoe (`if #available(macOS 26)`
  onde couber). `glassEffect` nativo disponível.
- Comentários e strings de UI em **pt-BR**. Simplificações deliberadas marcadas
  com `// ponytail:`.
- **Nunca editar `Knobler.xcodeproj` à mão.** Arquivo novo em `Knobler/` → rodar
  `xcodegen generate` antes do build. `project.yml` não muda (sources por diretório).
- Núcleo de modelo só-`Foundation` com self-check `@main` sob `#if` (molde de
  `Reminders.swift`/`Pomodoro.swift`), pra entrar como lib no app sem conflito de `@main`.
- Conjunto de quiosque **validado**: `[.hideDock, .hideMenuBar, .disableAppleMenu,
  .disableForceQuit, .disableProcessSwitching, .disableSessionTermination]`.
- Nível da janela de shield: `CGShieldingWindowLevel()`.
- Commits: só quando o usuário aprovar (política do harness). Os passos "Commit"
  abaixo são pontos lógicos, executados no fim mediante aval.

### Task 2: Modelo `ScreenBreak` + self-check

**Files:**
- Create: `Knobler/Descanso.swift`

**Interfaces:**
- Consumes: `Schedule`, `ReminderClock` (de `Reminders.swift`), `protocol Scheduled` (Task 1).
- Produces: `struct ScreenBreak: Codable, Identifiable, Equatable, Scheduled` com `id, label, schedule, durationMinutes, enabled`; enum `BreakFreq` (8 modos) com `makeSchedule`/`load`; `func scheduleSummary() -> String`.

- [ ] **Step 1: Criar `Descanso.swift`** — modelo + resumo humano (reusa
`ReminderClock.humanCalendar`/`humanInterval`) + self-check `@main` sob
`#if DESCANSO_SELFCHECK`:

```swift
//  Descanso.swift — modelo do bloqueio forçado. Só Foundation (reusa Schedule/
//  ReminderClock de Reminders.swift). Self-check:
//    swiftc -parse-as-library -swift-version 5 -D DESCANSO_SELFCHECK \
//      Knobler/Reminders.swift Knobler/Descanso.swift -o /tmp/dck && /tmp/dck
import Foundation

struct ScreenBreak: Codable, Identifiable, Equatable, Scheduled {
    var id: UUID = UUID()
    var label: String = ""            // opcional; overlay usa fallback "descanse um pouco"
    var schedule: Schedule
    var durationMinutes: Int = 5      // 1–120
    var enabled: Bool = true

    func scheduleSummary(cal: Calendar = ReminderClock.calendar) -> String {
        let base: String
        switch schedule {
        case .oneShot(let d):
            let f = DateFormatter(); f.calendar = cal; f.timeZone = cal.timeZone
            f.locale = Locale(identifier: "pt_BR"); f.dateFormat = "d 'de' MMM, HH:mm"
            base = "Uma vez · \(f.string(from: d))"
        case .interval(let m): base = "A cada \(ReminderClock.humanInterval(m))"
        case .calendar(let c): base = ReminderClock.humanCalendar(c)
        }
        return "\(base) · \(durationMinutes)min de bloqueio"
    }
}
```

- [ ] **Step 2: Adicionar o self-check** ao mesmo arquivo (assert-based, molde do
Pomodoro/Reminders): Codable roundtrip; "Daqui a X" → oneShot futuro; duração no
resumo. Ex.:

```swift
#if DESCANSO_SELFCHECK
@main enum DescansoSelfCheck {
    static func main() {
        var cal = Calendar(identifier: .gregorian)
        cal.timeZone = TimeZone(identifier: "America/Sao_Paulo")!
        ReminderClock.calendar = cal
        // Codable roundtrip
        let s = ScreenBreak(label: "Almoço",
                            schedule: .calendar([DateComponents(hour: 12, minute: 30)]),
                            durationMinutes: 15)
        let data = try! JSONEncoder().encode([s])
        assert(try! JSONDecoder().decode([ScreenBreak].self, from: data) == [s])
        // resumo inclui a duração
        assert(s.scheduleSummary(cal: cal) == "Todo dia · 12:30 · 15min de bloqueio")
        assert(ScreenBreak(schedule: .interval(minutes: 90))
                .scheduleSummary(cal: cal) == "A cada 90min · 5min de bloqueio")
        // "Daqui a X" é oneShot no futuro (mapeamento fica na UI; aqui só valida o shape)
        let future = Date().addingTimeInterval(3000)
        if case .oneShot(let d) = Schedule.oneShot(future) { assert(d == future) }
        print("descanso self-check ok")
    }
}
#endif
```

**Files:**

**Interfaces:**
- Produces: `final class BreakOverlayModel: ObservableObject { @Published var endDate: Date; @Published var label: String; @Published var holdProgress: Double }`; `struct BreakOverlayView: View` que recebe `@ObservedObject var model: BreakOverlayModel`.

- [ ] **Step 1: Criar `DescansoView.swift`** com o modelo observável e a view (fundo
0.9, contador mm:ss / h:mm:ss, rótulo com fallback, rodapé fixo "segurar Esc para
sair" + barra de hold). Segue `DESIGN.md` (cores `ink*`, SF Pro/SF Mono):

```swift
import SwiftUI

final class BreakOverlayModel: ObservableObject {
    @Published var endDate = Date()
    @Published var label = ""
    @Published var holdProgress = 0.0   // 0..1 enquanto segura Esc
}

struct BreakOverlayView: View {
    @ObservedObject var model: BreakOverlayModel
    // timer de UI: recomputa o restante do endDate (sleep-proof), não decrementa
    @State private var now = Date()
    private let tick = Timer.publish(every: 0.25, on: .main, in: .common).autoconnect()

    private var remaining: TimeInterval { max(0, model.endDate.timeIntervalSince(now)) }
    private var clock: String {
        let s = Int(remaining.rounded())
        return s >= 3600 ? String(format: "%d:%02d:%02d", s/3600, (s%3600)/60, s%60)
                         : String(format: "%02d:%02d", s/60, s%60)
    }
    private var subtitle: String {
        model.label.isEmpty ? "descanse um pouco" : model.label
    }

    var body: some View {
        ZStack {
            Color.black.opacity(0.9).ignoresSafeArea()  // ponytail: único knob de "quão escuro"
            VStack(spacing: 10) {
                Text(clock)
                    .font(.system(size: 72, weight: .light, design: .rounded))
                    .monospacedDigit().foregroundStyle(.white)
                Text(subtitle)
                    .font(.system(size: 15)).foregroundStyle(.white.opacity(0.6))
            }
            VStack {
                Spacer()
                VStack(spacing: 6) {
                    if model.holdProgress > 0 {
                        ProgressView(value: model.holdProgress)
                            .frame(width: 160).tint(.white.opacity(0.8))
                    }
                    Text("segurar Esc para sair")
                        .font(.system(size: 9, weight: .medium, design: .monospaced))
                        .foregroundStyle(.white.opacity(0.45))
                }
                .padding(.bottom, 40)
            }
        }
        .onReceive(tick) { now = $0 }
    }
}
```

### Task 5: `DescansoController` — janelas de shield + quiosque + escape

**Files:**
- Create: `Knobler/DescansoController.swift`

**Interfaces:**
- Consumes: `BreakOverlayModel`, `BreakOverlayView` (Task 4).
- Produces: `final class DescansoController { private(set) var isActive: Bool; var onEnd: (() -> Void)?; func begin(label: String, duration: TimeInterval); func abort() }`.

- [ ] **Step 1: Criar `DescansoController.swift`.** `begin` ignora se `isActive`;
computa `endDate`; monta uma janela por `NSScreen` no nível de shield; ativa o app
(`.regular` + `activate`) e liga o quiosque; instala o monitor de Esc; timer de
0.25s recomputa restante e `holdProgress`. `end`/`abort` faz fade-out, fecha,
restaura policy/presentationOptions, `isActive=false`, `onEnd?()`. Pontos-chave
(demais detalhes na `SPEC-descanso.md`):

```swift
import AppKit
import SwiftUI

final class DescansoController {
    private(set) var isActive = false
    var onEnd: (() -> Void)?

    private let model = BreakOverlayModel()
    private var windows: [NSWindow] = []
    private var timer: Timer?
    private var keyMonitor: Any?
    private var endDate = Date()
    private var escDownSince: Date?
    private static let kiosk: NSApplication.PresentationOptions =
        [.hideDock, .hideMenuBar, .disableAppleMenu,
         .disableForceQuit, .disableProcessSwitching, .disableSessionTermination]

    func begin(label: String, duration: TimeInterval) {
        guard !isActive else { return }          // ponytail: um lock por vez
        isActive = true
        endDate = Date().addingTimeInterval(duration)
        model.endDate = endDate; model.label = label; model.holdProgress = 0

        for screen in NSScreen.screens {
            let w = NSWindow(contentRect: screen.frame, styleMask: [.borderless],
                             backing: .buffered, defer: false)
            w.isOpaque = false; w.backgroundColor = .clear; w.hasShadow = false
            w.level = NSWindow.Level(rawValue: Int(CGShieldingWindowLevel()))
            w.collectionBehavior = [.canJoinAllSpaces, .fullScreenAuxiliary, .stationary]
            w.ignoresMouseEvents = false
            w.contentView = NSHostingView(rootView: BreakOverlayView(model: model))
            w.alphaValue = 0
            w.setFrame(screen.frame, display: true)
            w.orderFrontRegardless()
            windows.append(w)
        }
        // ativa + quiosque (as flags só valem com o app ativo; ver Achados)
        NSApp.setActivationPolicy(.regular)
        NSApp.activate(ignoringOtherApps: true)
        windows.first?.makeKeyAndOrderFront(nil)
        NSApp.presentationOptions = Self.kiosk

        NSAnimationContext.runAnimationGroup { ctx in
            ctx.duration = 0.4; windows.forEach { $0.animator().alphaValue = 1 }
        }
        installEscMonitor()
        let t = Timer(timeInterval: 0.25, repeats: true) { [weak self] _ in self?.tickUI() }
        RunLoop.main.add(t, forMode: .common); timer = t
    }

    private func tickUI() {
        if Date() >= endDate { end(); return }
        if let since = escDownSince {
            model.holdProgress = min(1, Date().timeIntervalSince(since) / 5)
            if model.holdProgress >= 1 { abort() }
        }
    }

    private func installEscMonitor() {
        keyMonitor = NSEvent.addLocalMonitorForEvents(matching: [.keyDown, .keyUp]) {
            [weak self] e in
            guard let self, e.keyCode == 53 else { return e }   // 53 = Esc
            if e.type == .keyDown { if self.escDownSince == nil { self.escDownSince = Date() } }
            else { self.escDownSince = nil; self.model.holdProgress = 0 }
            return nil                                          // engole o Esc
        }
    }

    func abort() { end() }   // v1: abort == fim antecipado; oneShot já se auto-desliga no onFire

    private func end() {
        guard isActive else { return }
        timer?.invalidate(); timer = nil
        if let m = keyMonitor { NSEvent.removeMonitor(m); keyMonitor = nil }
        escDownSince = nil
        NSApp.presentationOptions = []
        NSApp.setActivationPolicy(.accessory)
        NSAnimationContext.runAnimationGroup({ ctx in
            ctx.duration = 0.4; windows.forEach { $0.animator().alphaValue = 0 }
        }, completionHandler: { [weak self] in
            self?.windows.forEach { $0.orderOut(nil) }; self?.windows.removeAll()
        })
        isActive = false
        onEnd?()
    }
}
```
